\begin{abstract}
\noindent
This thesis presents AMATAS (Advanced Multi-Agent Threat Analysis System), a network intrusion detection architecture that combines a machine learning pre-filter with a six-agent large language model pipeline to deliver both accurate detection and transparent reasoning. Unlike traditional classifiers that produce opaque probability outputs, AMATAS generates multi-perspective reasoning chains from specialist agents analysing protocol conformance, statistical anomalies, behavioural signatures, and temporal patterns, with an adversarial Devil's Advocate mechanism for false positive control.

Evaluated across all fourteen attack types in the CICIDS2018 NetFlow v3 dataset at realistic class distributions (five per cent attack prevalence), AMATAS achieves a mean F1 score of 92.9 per cent across thirteen detectable attack categories, with a false positive rate of zero per cent in ten of fourteen experiments. A two-tier architecture combining a Random Forest pre-filter with the LLM pipeline reduces analysis cost by 94.6 per cent, from an estimated \$509.78 to \$27.35 for the complete evaluation corpus of 14,000 flows.

A controlled ablation study demonstrates that the multi-agent architecture outperforms the best single-agent configuration by 25.2 F1 percentage points, and that external threat intelligence tools provide no measurable benefit on anonymised datasets. The Infiltration attack category produces zero per cent recall, exposing a fundamental limitation of individual flow-level analysis that motivates the proposed temporal clustering extension.
\end{abstract}
